# playlist-debut
This is repo contains scripts for adding a focus song to a clean streams playlist.

`settings.py` structure for the code to work:
```
# Credentials
CLIENT_ID = ""
CLIENT_SECRET = ""
REDIRECT_URI = "http://localhost:3000"
# Focus song ID:
FOCUS_SONG_ID = ""

# Playlists:
PLAYLISTS_PREDEBUT = ['']

PLAYLISTS_DEBUT = ['']

HENAR_CHARTS_DEBUT = ['']

PLAYLISTS_TO_CLEAN = ['']
```
